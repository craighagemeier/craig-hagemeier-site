# Craig Hagemeier Portfolio

## Quick Overview (TL;DR)

Personal digital anchor point linking all my professional and creative profiles online.  
Showcases my frontend, design system, and UI skills through live projects and experimentation.  

- **Live Site:** [craighagemeier.com](https://craighagemeier.com/)  
- **Key Highlights:**  
  - Multiple themes with light/dark mode  
  - Dynamic Photography page with masonry layout  
  - Contact form and interactive features  
  - Endurance Quests tracker  
  - Inspiration and curated links  
  - **Tech Stack:** Next.js, React, TypeScript, SCSS, API integrations, design tokens  

Built as a public showcase for my work, experimentation playground, and professional online anchor.

---

## Purpose

I built this portfolio site for three main reasons:

1. **Digital Anchor:** When my name is searched online, this site appears first, providing a cohesive view of my professional and creative work.  
2. **Skill Showcase:** Since much of my work at United Airlines is proprietary, this site allows me to demonstrate my frontend, design system, and UI development skills publicly.  
3. **Experimentation Playground:** I use this site to explore new technologies, layouts, and UX patterns in a real-world project context.

---

## Key Features

- **Multiple Themes and Dark Mode:** Showcases theming and token-based design flexibility.  
- **Photography Page:** Pulls images from an API and displays them in a masonry layout. A challenging layout to implement correctly.  
- **Contact Page:** Fully submitable form to demonstrate handling user input and interactions.  
- **Inspiration Page:** Curated links to design and development sources that influence my work.  
- **Endurance Quests Page:** Tracks personal progress in my endurance sports goals.  
- **Responsive and Modern UI:** Works across devices, demonstrating frontend adaptability and attention to detail.  

---

## Technical Decisions

- **Next.js:** React framework with hybrid rendering. Most components are client-side (`"use client"`), with routing, static export, and server-side features handled by Next.js.
- **Theming and Tokens:** Implemented multiple themes to demonstrate how design tokens can drive flexible UI changes.  
- **Dynamic API Integration:** Photography page pulls content from an API, showing ability to handle asynchronous data.  
- **Masonry Layout:** Implemented custom masonry layout to demonstrate complex UI composition.  
- **Forms and Interactions:** Contact page form demonstrates handling inputs and basic form validation.
